# 座位编码参考

## 座位类型编码

| 名称 | 编码 | 说明 |
|------|------|------|
| 二等座 | O | 最常见，高铁/动车标准座位 |
| 一等座 | M | 更宽敞，2+2 座位布局 |
| 商务座 | 9 | 最高级，1+2 或 1+1 布局 |
| 特等座 | P | 部分车次提供 |
| 硬座 | 1 | 普通列车 |
| 硬卧 | 3 | 普通列车卧铺 |
| 软卧 | 4 | 普通列车卧铺 |

用于 `order.ts --seat` 参数。

## 选座位置

二等座/一等座车厢的座位字母分布：

| 字母 | 位置 | 适用于 |
|------|------|--------|
| A | 靠窗，左侧 | 二等座、一等座 |
| B | 中间，左侧 | 仅二等座（3+2 布局） |
| C | 过道，左侧 | 二等座、一等座 |
| D | 过道，右侧 | 二等座、一等座 |
| F | 靠窗，右侧 | 二等座、一等座 |

用户意图映射：
- "靠窗" → A 或 F
- "过道" → C 或 D

## order.ts --seat-position 格式

`{乘客序号}{座位字母}`，序号从 1 开始。

示例：
- 单乘客靠窗：`1F`
- 双乘客，第一人靠窗第二人过道：`1F2D`
- 不选座：省略 `--seat-position` 参数
